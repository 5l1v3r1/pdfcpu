# Fonts

Print list of supported fonts.

## Usage

```
usage: pdfcpu fonts
```

<br>


## Example

```sh
pdfcpu fonts
Courier
Courier-Bold
Courier-BoldOblique
Courier-Oblique
Helvetica
Helvetica-Bold
Helvetica-BoldOblique
Helvetica-Oblique
Symbol
Times-Bold
Times-BoldItalic
Times-Italic
Times-Roman
ZapfDingbats
```
